# Chris Taschenrechner

Dieses Projekt ist ein einfacher Taschenrechner, der in HTML, CSS und JavaScript geschrieben ist.

## Dateien

Das Projekt besteht aus drei Hauptdateien:

1. `Taschenrechner.html`: Dies ist die Haupt-HTML-Datei, 
    die die Struktur des Taschenrechners definiert. Sie enthält auch Links zu den CSS- und JavaScript-Dateien.

2. `Taschenrechner.css`: Diese Datei enthält alle Stile, die für den Taschenrechner benötigt werden. 
    Sie definiert das Layout und das Aussehen des Taschenrechners, einschließlich der Größe und Position der Tasten und des Displays, der Hintergrundfarben und der Schriftgrößen.

3. `Taschenrechner.js`: Dies ist die JavaScript-Datei, die die Funktionalität des Taschenrechners bereitstellt. 
    Sie enthält Code zum Hinzufügen von Event-Listenern zu den Tasten des Taschenrechners, zum Durchführen von 
    Berechnungen und zum Aktualisieren des Displays.

## Verwendung

Einfach die 3 Dateien (html, css und js ) in einem Ordner runter laden und

Öffnen Sie einfach die `Taschenrechner.html`-Datei in einem Webbrowser, um den Taschenrechner zu verwenden.

## Erklährungen

*html:

im code sind die einzelnen Zeilen beschrieben nur kurz ein paar wörter zum <div>....</div>

Das <div>-Element in HTML steht für “division” und wird verwendet, um einen Abschnitt oder eine Gruppe von Elementen 
auf einer Webseite zu umschließen. Es ist ein Block-Level-Element und wird oft zusammen mit 
CSS verwendet, um das Layout oder die Formatierung der umschlossenen Elemente zu steuern. 
Es hat keine spezielle Bedeutung an sich, sondern dient dazu, andere HTML-Elemente zu gruppieren und zu formatieren.

*javascript:

hier ein paar stichwörtererklärungen:

Array: Ein Array ist ein globaler Objekttyp in JavaScript, der eine Liste von Elementen speichert. Sie können auf die Elemente eines Arrays zugreifen, indem Sie ihre Indexnummer verwenden, die bei 0 beginnt.

switch: Die switch Anweisung ist eine Kontrollstruktur, die es ermöglicht, mehrere case Anweisungen zu überprüfen und auszuführen. Es wird oft als sauberere Alternative zu langen if...else if...else Anweisungen verwendet.

case: Innerhalb einer switch Anweisung repräsentiert eine case Anweisung einen möglichen Pfad, den der Code ausführen kann. Jede case wird mit einem Wert oder Ausdruck verglichen und wenn eine Übereinstimmung gefunden wird, wird der Code innerhalb dieser case Anweisung ausgeführt.

break: Die break Anweisung wird verwendet, um die Ausführung einer Schleife oder einer switch Anweisung sofort zu beenden und zur nächsten Anweisung nach der Schleife oder switch zu springen.

try/catch: Die try/catch Anweisungen werden verwendet, um Fehler in JavaScript zu behandeln. Der Code innerhalb des try Blocks wird ausgeführt und wenn ein Fehler auftritt, wird die Ausführung an den catch Block übergeben, der den Fehler behandelt.

default: Innerhalb einer switch Anweisung repräsentiert die default Anweisung einen Pfad, der ausgeführt wird, wenn keine der case Anweisungen übereinstimmt. Es ist ähnlich wie das else in einer if...else if...else Struktur.
